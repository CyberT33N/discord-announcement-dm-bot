Ubuntu Server

Currently only Warranty & Support for the Ubuntu Server with Desktop! In order to install the Desktop please do the following steps:

1. Install Ubuntu Server as example on Virtualbox via the .iso from above

4. Then select Ubuntu Gnome Desktop! You can mark the boxes with the SPACE key
5. Press ENTER to confirm your changes

6. After everything was installed please use: sudo reboot

Support





Click here to show more details..



If you ever face a bug / something is not correctly scrapped then you MUST do the following steps:


1. (Windows only)
Increase your Terminal Buffer Size to the max value that your Windows Terminal Window will in general displays more Logs Text!

The default Terminal Buffer Size is really small. Please make right click on the Top Border of your Terminal Window and then click Settings.



After this go to Options and increase Buffer Size to 999







2. YOU MUST send me the Terminal Logs from the Terminal Window (start.bat / .sh Process)

Exactly a few seconds later after the problem happen or when the bot crashed you YOU MUST copy the complete terminal text. It´s almost same on every OS...



Windows:

You mark the text with your mouse and press ENTER to copy the text or use right click to copy the text.

Or you can press aswell CTRL + A like normally to copy all text from the terminal and press ENTER to copy the text or use right click to copy the text.

Your logs will be now in your clipboard and can be imported to a text editor to save it as .txt file as example.

IMPORTANT
When the Bot in general will crash on Windows then the Terminal Window will be closed when you press a key and then all your Log Text will be lost!



Please make always sure to read the last line of the Terminal Window and search for text like Press a key to exit. If you see this text please use your left mouse click instead by holding it to mark the complete Terminal Window Text by scrolling from the bottom to the top while you marking the text!

Then on the top of the Terminal Window when you reach the first line press ENTER to copy it!

Your logs will be now in your clipboard and can be imported to a text editor to save it as .txt file as example.







MAC:

You mark the text with your Mouse Left Click by holding it and mark everything from the bottom to the top or press CTRL + A to mark everything directly

After this press Mouse Right Click and click on copy




Linux:

You mark the text with your Mouse Left Click by holding it and mark everything from the bottom to the top

After this press Mouse Right Click and click on copy

VERY IMPORTANT

When the Bot will restart itself with the than we will use gnome-terminal to open a new terminal window. The problem is that this new Terminal Window will for default get closed and this means all Logs will get lost!

In gnome-terminal, go to Edit -> Profile Preferences -> Title. Click the Command tab. Select Hold the terminal from the drop-down menu labelled When command exits. You should change this for your default profile which you will use with all Terminal Windows!

FAQ





Click here to show more details..



Here you will find the most important questions you may want to know..


• How do I add the Bot to my Server?


Please copy the Client ID from your Application General Information Section and paste it to the link from below:
https://discordapp.com/oauth2/authorize?client_id=xxxxxxxxxxx&scope=bot

• My Bot commands will be currently used from all my Channels in my Server. How do I allow it only for one specific Channel?


You must create Channel Roles for your Bot.
This means on all channels where you don´t want that people can use the commands you will click on the wheel next to the Channel Name and then go to Permissions.

Create a new role for your Bot and block Read Messages & Send Messages by clicking on the red X!

Please click on save after change.

• How can I manually set a User Role in my Channel?


You must go the Server Settings and create Roles.

Then inside of your Server Chat please go inside of a Channel and then left click on a User and select "add role"!

You will see now a menu with your previous created Roles which you can use now.
